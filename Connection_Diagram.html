NUCLEO-F401RE — Physical Connection Guide

Car Simulator · ECU1 (Control System) + ECU2 (Actuator System) · UART D8/D2 Link
STM32F401RE Nucleo-64 UART 57600 baud on D8/D2 Code confirmed compatible IRremote v3 library
✅ Code Compatibility: Yes — the code is fully compatible with this exact board (NUCLEO-F401RE / NUF401RESAU1). All pins used exist on this board's CN5/CN6/CN7/CN10 headers. The board you photographed is correct.
01

UART Link — 3 Wires Between Both Boards

CROSS-CONNECT the two Nucleo boards exactly like this:

ECU1 D8 (PA9) TX ──────→ ECU2 D2 (PA10) ECU1 transmits → ECU2 receives
ECU1 D2 (PA10) RX ←────── ECU2 D8 (PA9) ECU2 transmits → ECU1 receives
ECU1 GND GND ────── ECU2 GND ⚠ CRITICAL — NO GND = NO UART
📍 Physical location on board: D8 and D2 are on the RIGHT-SIDE header (CN10) of the Nucleo board — the long connector on the right when USB is at the top-left. D8 is near the middle, D2 is near the top-right of that header.

⚠ Ultrasonic ECHO voltage divider (REQUIRED on ECU1):
HC-SR04 ECHO pin outputs 5V. STM32F401RE GPIO is 3.3V max.

Step-by-Step Component Wiring

🔌L298N Motor Driver → ECU2
12V pin←12V battery positive
GND pin←Battery GND + STM32 GND (shared)
ENA pin←ECU2 D9 (PA15) — PWM speed A
IN1 pin←ECU2 D4 (PA5)
IN2 pin←ECU2 D12 (PA6)
ENB pin←ECU2 D3 (PB3) — PWM speed B
IN3 pin←ECU2 D11 (PA7)
IN4 pin←ECU2 A3 (PB0)
OUT1/OUT2→Motor 1 (rear left wheel)
OUT3/OUT4→Motor 2 (rear right wheel)
⚙️Servo Motors → ECU2
Servo1 Signal←ECU2 A0 (PA0) — Steering
Servo1 VCC←5V external supply
Servo1 GND←GND (shared)
Servo2 Signal←ECU2 A1 (PA1) — Wiper
Servo2 VCC←5V external supply
Servo2 GND←GND (shared)
Note→Never power servos from STM32 5V pin — use external 5V/1A supply
📻IR Receiver TSOP1738 → ECU1
Pin 1 (OUT)→ECU1 D7 (PA8)
Pin 2 (GND)→GND
Pin 3 (VCC)→3.3V + 100µF cap across VCC-GND
First run→Open ECU1 Serial Monitor, press RED/GREEN buttons, note hex codes printed, update IR_CODE_LOCK and IR_CODE_UNLOCK in ECU1 sketch
📡HC-SR04 Ultrasonic → ECU1
VCC→5V pin on Nucleo
GND→GND
TRIG→ECU1 D6 (PB10) — direct wire
ECHO→1kΩ → ECU1 D5 (PB8) → 2kΩ → GND
⚠ WHY→ECHO = 5V output. STM32 GPIO = 3.3V max. Divider is mandatory or you will damage the MCU
🕹️Joysticks (×2) → ECU1
Both VCC→3.3V on Nucleo
Both GND→GND
Joy1 VRx→ECU1 A0 (PA0) — Steering
Joy1 SW→ECU1 A2 (PA4) — Horn (other end GND)
Joy2 VRx→ECU1 A1 (PA1) — Accel/Brake
Pot wiper→ECU1 A3 (PB0) — Wiper speed
🔲Push Buttons (×3) → ECU1
BTN1 pin A→ECU1 D3 (PB3) — Fwd/Rev
BTN1 pin B→GND
BTN2 pin A→ECU1 D4 (PB4) — Parking
BTN2 pin B→GND
BTN3 pin A→ECU1 D5 (PB5) — Headlight
BTN3 pin B→GND
No resistor→INPUT_PULLUP used in code — no external resistor needed
🔢TM1637 7-Seg Displays (×2) → ECU2
Spd CLK→ECU2 D9 (PC7)
Spd DIO→ECU2 D6 (PA11) — shared
RPM CLK→ECU2 D13 (PA12)
RPM DIO→ECU2 D6 (PA11) — same shared wire
Both VCC→3.3V or 5V
Both GND→GND
🟡LED Bar Graph (Parking, 6 LEDs) → ECU2
LED 1 anode←330Ω ← PC4 (use Morpho header)
LED 2 anode←330Ω ← PC5
LED 3 anode←330Ω ← PC8
LED 4 anode←330Ω ← PC9
LED 5 anode←330Ω ← PC10
LED 6 anode←330Ω ← PC11
All cathodes→GND
Note→PC4–PC11 are on the MORPHO connector (inner rows CN7/CN10) — use female-to-male jumpers
💡Brake LED + Headlight Relay → ECU2
Brake LED+←220Ω ← ECU2 D6 (PC2) — PWM dims it
Brake LED-→GND
Relay IN←ECU2 D5 (PC3)
Relay VCC←5V
Relay GND→GND
Relay COM←12V Battery+
Relay NO→Headlight bulb +
Bulb -→GND (battery -)
🔊Buzzers (×2) → ECU2
Horn BUZ+←ECU2 D7 (PA8)
Horn BUZ-→GND
Indic BUZ+←ECU2 D10 (PC6 morpho)
Indic BUZ-→GND
Type→Must be PASSIVE buzzers (not active) — tone() needs passive type
